# مقيّم الألعاب التعليمية 🧩

وكيل ذكاء اصطناعي متخصص لتقييم الألعاب التعليمية وفق منهج مونتيسوري ومعايير السلامة الأوروبية EN 71.

## المميزات

- **تقييم مونتيسوري**: تحليل شامل لمدى توافق اللعبة مع مبادئ منهج مونتيسوري
- **معايير EN 71**: تقييم مبدئي لمعايير السلامة الأوروبية (EN 71-1, EN 71-2, EN 71-3)
- **مقارنة لعبتين**: تحليل مقارن مفصّل بين لعبتين تعليميتين
- **تقرير شامل**: تقرير احترافي يتضمن جداول تقييم وتوصيات
- **تصدير PDF**: تصدير التقرير بصيغة PDF
- **تصدير Word**: تصدير التقرير بصيغة Word (.docx) مع دعم RTL

## متطلبات التشغيل

- Node.js 18+
- مفتاح Anthropic API

## التثبيت

```bash
cp .env.example .env.local
# أضف مفتاح API في .env.local
npm install
npm run dev
```

ثم افتح: http://localhost:3000

## طريقة الاستخدام

1. اختر نوع التقييم: لعبة واحدة أو مقارنة لعبتين
2. ارفع صورة (صور) اللعبة
3. أدخل اسم اللعبة (اختياري)
4. انقر **تحليل وتقييم اللعبة**
5. صدّر التقرير بصيغة PDF أو Word

## تنبيه

التقييمات مبنية على تحليل الصور ولا تغني عن الاختبارات المعملية الرسمية لمعايير EN 71.
